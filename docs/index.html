ErrandHelper Frontend Chat System Documentation

🚀 Overview

The ErrandHelper frontend chat system is built using Angular and provides real-time messaging capabilities through WebSocket connections. It integrates seamlessly with the Django backend to deliver a complete chat experience.

🏗️ Architecture

The chat system follows Angular's component-based architecture with clear separation of concerns:

  • Components: UI components for chat interface
  • Services: Business logic and API communication
  • Models: TypeScript interfaces and data structures
  • WebSocket: Real-time communication layer

🧩 Components

ChatService

Location: src/app/apps/components/messages/services/chat.service.ts

Key Methods:

WebSocketService

Location: src/app/apps/components/messages/services/websocket.service.ts

Key Features:

Key Methods:

🔌 API Integration

The frontend communicates with the Django backend through multiple channels:

REST API Endpoints

  • GET /api/chat/rooms/ - List chat rooms
  • POST /api/chat/rooms/ - Create new chat room
  • GET /api/chat/messages/{room_id}/ - Get messages for room
  • POST /api/chat/messages/ - Send new message
  • POST /api/chat/upload/ - Upload file attachments

WebSocket Events

  • message.new - New message received
  • message.delivered - Message delivery confirmation
  • message.read - Message read receipt
  • user.typing - User typing indicator
  • user.joined - User joined chat room
  • user.left - User left chat room

Authentication

  • JWT token-based authentication
  • Automatic token refresh
  • Secure WebSocket connection with auth headers
  • Role-based access control

⚡ Setup & Configuration

Installation Steps

1. Install dependencies: npm install
2. Configure environment variables:
   - API_BASE_URL: Backend API endpoint
   - WEBSOCKET_URL: WebSocket server URL
   - JWT_SECRET: JWT token secret

3. Update environment files:
   - src/environments/environment.ts
   - src/environments/environment.prod.ts

4. Run development server: ng serve

Configuration Files

  • src/environments/environment.ts - Development environment
  • src/environments/environment.prod.ts - Production environment
  • angular.json - Angular CLI configuration
  • tsconfig.json - TypeScript configuration
